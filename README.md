# 网络面板 (NetworkPanel)

在线网络测速工具，实时监测您的网络情况。

## 功能特性

- 网速测试
- 网络环境监测
- 多测试节点
- PWA 支持（可添加到桌面）
- 浅色/深色主题自动切换

## 部署

本项目为构建后的静态文件，可直接部署到 GitHub Pages、Vercel、Netlify 等静态托管服务。

### GitHub Pages 部署

1. 进入仓库设置 (Settings)
2. 找到 Pages 选项
3. Source 选择 "Deploy from a branch"
4. Branch 选择 `main`，文件夹选择 `/ (root)`
5. 点击 Save

或使用自动 CI 部署（推送到 main 分支自动部署）。

## 本地运行

由于本仓库仅包含构建后的静态文件，如需本地开发需要源代码。
